### How to export these from Power BI
1. Open `data/EcoDriveMotors_Dashboard.pbix` in **Power BI Desktop**.
2. `File → Export → Export to PDF`, then convert each PDF page to a PNG (e.g. with any free PDF-to-image tool), **or**
3. Use the live published report: open each page in the browser, maximize the view (top-right ⤢ icon), then take a screenshot.

Once added, reference them in the main `README.md`, e.g.:

```markdown
![KPI 1 — Sales & Revenue](./docs/images/03-kpi1-sales-revenue.png)
```
